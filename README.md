# Fleetops_Lite
A lightweight fleet management dashboard built with React and Flask.

## Features
- Vehicle dashboard UI
- Flask API backend
- Real-time status simulation (planned)
- CRUD operations (in progress)

## Tech Stack
- React (Frontend)
- Flask (Backend API)
- GitHub (Version control)

## Project Structure
frontend/ → React app  
backend/ → Flask API
